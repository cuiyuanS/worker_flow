微服务流传系统
=======
该用户界面将允许您使用基于图形、节点、流程图的界面设计和执行微服务流传系统
# 项目介绍

### 项目背景

- 微服务流传系统是一种基于图形、节点、流程图的界面设计和执行微服务流传系统

- 该系统使用Gearman作为任务调度器，

- 使用redis作为缓存，

- 使用MongoDB作为数据库，

- 使用Python作为后端语言，

- 使用vue作为前端框架，

- 使用docker作为容器化工具，
